Lumi · assessment app

A personality and skill assessment app — and a lesson in where systems complexity hides

Lumi is a personality and skill assessment app with Google login, personalised results, and a dashboard. The visible product is clean. The harder part was making authentication work reliably across server-rendered flows without silent failure — a class of bug where the system looks healthy on the surface until the next request proves otherwise.

In development Next.js Google OAuth Supabase Auth
What I built

An assessment flow with Google login, structured results, and a personalised dashboard

  • Google OAuth login flow handling both first-time sign-up and returning sessions correctly.
  • Assessment questions mapped to personality and skill dimensions with weighted scoring.
  • Results page presenting dimension scores with interpretation copy tailored per profile.
  • Personalised dashboard accessible after login, pulling the user's assessment history from Supabase.
Where it got hard

The dangerous bugs were the ones that looked like success

A helper designed for one runtime context silently swallowed the cookie write needed for OAuth callback handling. The auth flow completed without error. The session was written — but not in a form the next server-rendered request could read. The system looked healthy until the dashboard failed to recognise the session on the next page load.

The fix was not a patch — it was understanding which runtime context each helper was designed for and why that mattered for server-rendered flows. Missing values and wrong values are different failure modes. Fallback logic handles one of them. This was the other.

What it shows

Product instinct applied to system-level debugging

Lumi is useful in the portfolio not as the most ambitious build, but as evidence of how I approach unexpected system behaviour: trace to root cause, understand the class of problem, fix the underlying condition rather than the visible symptom. When a bug chain gets long, the visible error is almost always downstream from the real cause.
