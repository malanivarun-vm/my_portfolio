How deep should each project go?

The projects need to show your thinking — problem → approach → outcome. The question is where and how.

A

Expand in place

Cards stay compact. Clicking "See how I approached this" expands the card inline to reveal the full thinking — problem, decisions, outcome, numbers. No separate pages.

B

Structured sections per project

Each project card is longer on the main page — showing Problem / Approach / Outcome as distinct labelled sections. Everything visible without clicking. Richer, but takes more space.

C

Separate case study pages

The main page shows a teaser card. Each project links to its own full-page case study with complete breakdown, context, and impact. Highest depth, most effort to write.
